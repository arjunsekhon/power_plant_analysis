# Power Plant Analysis

## Getting started

Clone the repo, activate virtual environment, install dependencies and run jupyter:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
jupyter lab
```
